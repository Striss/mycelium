What is this?

MYCELIUM is a website that grows itself. Each night, a cron job runs a genetic mutation algorithm that evolves the organism's traits, then calls an AI to generate a completely unique HTML page embodying those traits.


Over time, watch the organism's personality drift across aesthetic space. Once a month, an extinction event resets the genome entirely — all memory lost. Pages shaped by visitor votes are marked HIVE.
